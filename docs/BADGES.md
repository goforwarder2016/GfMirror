## How to Add Badges

1. Copy the badge markdown code
2. Paste it in your README.md file
3. Replace `yourusername` with your actual GitHub username
4. Update the repository name if different
5. Customize colors and text as needed

## Badge Customization

You can customize badges by:
- Changing colors using hex codes
- Adding custom text
- Using different styles (flat, plastic, etc.)
- Adding logos from [Simple Icons](https://simpleicons.org/)

Example:
```markdown
![Custom Badge](https://img.shields.io/badge/Custom-Text-ff0000?style=flat&logo=github&logoColor=white)
```
